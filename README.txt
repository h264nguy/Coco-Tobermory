SUMMER CAMP MEMORIES WEBSITE

How to use:
1. Open index.html in your browser to preview the website.
2. Replace the sample image links with your real photos.
3. To use local photos, create a folder named photos and change image URLs like this:
   photos/photo1.jpg
4. Change the Google Drive upload link in index.html under the Share Photos section.
5. Upload index.html, style.css, and script.js to GitHub Pages.

Main sections:
- Home
- Our Story
- Trip Highlights
- Photo Gallery
- Share Photos

Tip:
For real group sharing, create a Google Drive folder, set permission to anyone with the link can upload/edit, then replace the Upload button link.
